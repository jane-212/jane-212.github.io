# Jane's Blog

## Helix配置文件

```toml
theme = "gruvbox_dark_soft"

[editor]
line-number = "relative"
cursorline = true

[editor.auto-save.after-delay]
enable = true
timeout = 500

[editor.indent-guides]
render = true
```
